Download for macOS View on GitHub

Why this exists

npm supply-chain attacks have doubled in 2025-2026. The PolinRider/openclaw campaign hit hundreds of public repos, force-pushing malicious payloads from compromised dev machines for days before detection.

📦

Postinstall scripts

npm runs arbitrary code from any package you install. The #1 attack vector.

🪤

Config injection

Malware silently appends payloads to postcss.config, tailwind.config, etc.

🔥

Mass force-push

Once it has your git creds, it overwrites every branch on every repo you can push to.

🔓

.env exfiltration

Removes .env* from .gitignore so any committed env file leaks secrets.

Five-layer defense

Each layer blocks a different stage. Even if 4 fail, the 5th catches it.

1

Block npm install scripts

Sets ignore-scripts=true in ~/.npmrc. Stops postinstall malware before it ever runs.

2

DNS-block C2 servers

Adds known attacker domains to /etc/hosts. If reinfected, can't phone home.

3

Git pre-commit hook

Blocks commits containing malware signatures. Even infected files can't reach your remote.

4

Force-push watcher

Polls GitHub hourly. macOS notification on any unexpected force-push.

5

Daily malware scan

Searches your Desktop at 9am. Catches indicators that slip through other layers.
